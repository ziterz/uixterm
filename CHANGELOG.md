## 0.1.0 - Initial Release

### Added

- Initial UiXTerm library structure.
- ANSI foreground color support.
- ANSI background color support.
- ANSI text style support.
- ANSI reset functionality.
- Helper methods for formatted text output.
- Public library entry point through `uixterm.dart`.
- Example application demonstrating usage.
- Initial unit tests.
- Static analysis configuration.
