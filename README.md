# UiXTerm

ANSI Terminal Formatting Library for Dart.

UiXTerm is a lightweight Dart library designed to simplify ANSI terminal formatting by providing an organized API for:

- Foreground colors.
- Background colors.
- Text styles.
- Reset controls.
- Formatted text helpers.

The goal of UiXTerm is to make terminal interfaces and CLI applications easier to build using clean and readable Dart code.

---

## Features

Current features:

- ANSI foreground colors.
- ANSI background colors.
- Text formatting styles.
- Individual ANSI reset controls.
- Helper methods for formatted messages.
- Static API without object instantiation.
- Null-safe Dart implementation.

---

## Installation

Add UiXTerm to your project dependencies:

```yaml
dependencies:
  uixterm: ^0.1.0
```
