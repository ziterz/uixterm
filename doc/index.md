# UiXTerm Documentation

Welcome to the official documentation for **UiXTerm**, a modern ANSI terminal formatting library for Dart.

UiXTerm provides an intuitive API for styling terminal output using ANSI escape sequences. Instead of memorizing ANSI codes, you can use descriptive classes and constants to create colorful, readable, and maintainable console applications.

## Supported Platforms

UiXTerm is designed for modern ANSI-compatible terminals, including:

- Windows Terminal
- PowerShell
- Command Prompt (Windows 10/11 with Virtual Terminal support)
- Visual Studio Code Terminal
- Linux terminals
- macOS Terminal
- iTerm2
- SSH terminals
